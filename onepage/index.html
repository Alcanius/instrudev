About

Ingeniero de Software con sólida formación académica — Universidad Surcolombiana (2015-2017) e FET (2017-2020), complementada con un diplomado en Ciberseguridad (2019) — y más de cinco años de trayectoria diseñando, desarrollando y liderando proyectos de TI.

Ha desempeñado cargos de desarrollador, analista y líder de proyectos en sectores público y privado, dominando PHP Lenguaje backend , JavaScript Lenguaje web , React Biblioteca UI , Angular Framework web y Laravel Framework PHP , y gestionando equipos multidisciplinarios para cumplir objetivos de negocio.
